## Local deploy

Clone the project

```bash
  git clone https://github.com/thiago-srosa/youtube-animated-profile-card
```

Go to the project's directory

```bash
  cd youtube-animated-profile-card
```

Install the dependencies

```bash
  npm install
```

Start the local server

```bash
  npm run dev
```
